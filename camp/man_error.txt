SERIAL ERROR CODES:
	Serial error codes are generated when a UART commmunication flag is raised
heck the error codes below to troubleshoot

RADIO_FAIL                  there is no communication with the device. Check to see
					        if your device is connected. For more information use
				            the man test to run diagnostics.

NO_ACK

TRANSMISSION_FAILED
